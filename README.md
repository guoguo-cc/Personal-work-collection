# 项目

## **iConsolePro（智能健身App）**

   这是一款基于 Unity 的多人在线智能健身应用，支持通过 FTMS 蓝牙协议连接跑步机、划船机、动感单车等多种健身设备，实时解析并同步运动数据，完成速度、距离、卡路里、功率等
运动指标计算及 UI 可视化展示。

   项目包含蓝牙通信、设备数据解析、角色切换、运动数据云端存储、3D 场景渲染等核心模块，通过实时驱动 Animator 动画系统，实现虚拟角色与实体设备运动状态同步，为用户提供沉
浸式在线健身体验。



## **小娱大话骰（多人单机棋牌）**

   《小娱大话骰》是一款基于 Unity 开发的多人回合制棋牌类休闲游戏，以经典酒桌游戏“叫骰劈酒”为核心玩法，支持多人实时轮流叫骰、开骰判定等完整游戏流程，并兼容 Android 与 
WebGL 平台，为玩家提供轻量化的线上娱乐体验。



## **微笑方格（2D触控平台）**

   这是一款基于 Unity 开发的 2D 平台闯关游戏，玩家通过单点触控控制角色移动，利用墙体反弹机制完成关卡探索，收集宝藏并到达终点完成挑战。项目包含角色控制、物理碰撞、关卡选择、动画与粒子特效、数据存储等完整游戏系统，并采用模块化设计，便于后续关卡扩展与功能迭代。
